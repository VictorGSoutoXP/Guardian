# Security Policy

## Scope

Guardian AV is an educational security project and prototype. It is not intended to replace an enterprise or commercial antivirus product.

## Safe use guidelines

- Do not rely on this project as your only security control.
- Do not test detections with live malware on your main machine.
- Prefer controlled folders, disposable virtual machines, and non-production systems.
- Review generated reports before taking destructive actions.

## Reporting concerns

If you find a bug that could cause unsafe behavior, open an issue with:

- operating system and Python version
- exact command used
- error output or screenshot
- steps to reproduce

Avoid uploading harmful files directly to public issues.
